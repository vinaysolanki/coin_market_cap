# Coin Market Cap

Ruby gem to fetch Crypto currency prices from Coin Market Cap

## Installation

Add this line to your application's Gemfile:

    gem 'coin_market_cap'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install coin_market_cap

## Usage

### Price:

	bitcoin = CoinMarketCap::Coin.new('bitcoin')

	bitcoin.price
	#=> 847.63
